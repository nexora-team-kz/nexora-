# NEXORA

## Техническое задание (Версия 1.0)

### Главная цель
Создать AI-платформу для автовладельцев, которая помогает быстро находить запчасти, диагностировать неисправности и находить проверенные СТО.

## Основные функции

### 1. Поиск запчастей
- По названию.
- По VIN.
- По фотографии.
- По голосовому запросу.

### 2. AI-механик
- Анализ симптомов.
- Анализ фотографий.
- Анализ звука двигателя.
- Возможные причины неисправности.

### 3. Каталог СТО
- Проверенные сервисы.
- Отзывы.
- Рейтинг.
- Контакты.
- Построение маршрута.

### 4. Магазины запчастей
- Наличие.
- Цена.
- Аналоги.
- Доставка.

### 5. Личный кабинет
- Мои автомобили.
- История ремонтов.
- История заказов.
- Избранные магазины.
- Избранные СТО.
